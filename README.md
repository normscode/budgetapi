# BudgetApi

A simple **ASP.NET Core Web API** for **budget tracking** (income/expenses/categories).  
Built as a personal project to practice **C#/.NET**, RESTful APIs, EF Core, migrations, and basic service-layer structure.

---

## Features (Typical)

- CRUD endpoints for budget-related resources
- DTO-based request/response
- Service layer for business logic
- Database persistence using Entity Framework Core
- Migrations included


---

## Tech Stack

- **ASP.NET Core Web API**
- **Entity Framework Core**
- **SQL Server / LocalDB** 
- Swagger / OpenAPI
